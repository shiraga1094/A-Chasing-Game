# A Chasing Game
一個追逐遊戲，抑或是遊戲追逐你? </br>
A chasing game, or is the game chasing you?

##Content
- 一個簡單的追逐遊戲
- 獨具特色的多種能力
- 具有成就系統
- PlayerData紀錄玩家遊玩資訊
- 多樣化自訂功能
